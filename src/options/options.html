Settings

Save a GitHub token and the repositories you want to review. The dashboard fetches data when refreshed; the popup reads saved values.

GitHub fine-grained personal access token

This token is used to import repositories and refresh repository stats. For fine-grained tokens, select the repositories you want to monitor. Repository traffic, clones, and referrers require Administration: Read-only permission.

A token limited to selected repositories can only import and monitor those repositories.

The token is stored locally in Chrome extension storage and should not be committed or shared.

The connection test can verify repository data and traffic access separately.

Repositories

Use owner/repo or a GitHub repository URL. Values are saved as owner/repo. You can configure up to 20 repositories.

Settings

Save a GitHub token and the repositories you want to monitor. The dashboard fetches data; the popup only reads cached values.

GitHub fine-grained personal access token

To create a token, open GitHub, then go to Settings → Developer settings → Personal access tokens → Fine-grained tokens. Select the repositories you want to monitor, then grant repository Administration: Read-only permission so traffic stats can be read.

The token is stored locally in Chrome extension storage and should not be committed or shared.

The connection test can verify repository data and traffic access separately.

Repositories

Use owner/repo or a GitHub repository URL. Values are saved as owner/repo. You can configure up to 20 repositories.

Connection test

Test the token and visible repository rows before saving settings.
